# QuickGraphQL



# 简述



一个简单的工具来快速构建 GraphQL API 风格的通信方式，快捷简单的转换后端交互风格，而不需要深入了解GraphQL的具体构建细节，

低代码开发平台，自动生成scheme.graphql ，resolver ，provider，datafetcher，entity 等代码，使用注解 `@GraphQLEntity` 标注领域模型
